# 🚀 Exercícios de Lógica de Programação em Java

## 📅 Roadmap de Desenvolvimento e Progresso

O plano de estudos foi dividido em 4 níveis de complexidade, focando na consistência diária e na evolução gradativa dos conceitos de programação.

### 🟢 Nível 1: Estrutura Sequencial e Variáveis
*Foco em entrada de dados, operadores matemáticos básicos e manipulação de variáveis.*
- [x] `OlaMundo` - Mensagem de boas-vindas ao ecossistema.
- [x] `SomaSimples` - Recebe dois números inteiros e exibe a soma.
- [x] `MediaNotas` - Calcula a média aritmética de 4 notas.
- [x] `ConversorMoedas` - Converte um valor em Reais para Dólares com cotação fixa.
- [x] `CalculadoraSalario` - Calcula o salário mensal baseado em valor/hora e horas trabalhadas.
- [x] `ConversorTemperatura` - Converte graus Celsius para Fahrenheit.
- [x] `AreaRetangulo` - Calcula a área de um retângulo a partir da base e altura.
- [x] `DiasVividos` - Calcula a quantidade aproximada de dias vividos com base na idade.

### 🟡 Nível 2: Estruturas de Condição (if / else)
*Foco em tomadas de decisão baseadas em regras de negócio.*
- [x] `ParOuImpar` - Verifica se um número inteiro é par ou ímpar.
- [x] `MaiorDeIdade` - Valida se o usuário já atingiu a maioridade.
- [x] `AprovacaoEscolar` - Classifica o aluno entre Aprovado, Recuperação ou Reprovado.
- [x] `MaiorDeTres` - Compara três números e aponta o maior deles.
- [x] `CalculadoraBasica` - Executa operações matemáticas com base no operador escolhido.
- [x] `ValidadorTriangulo` - Verifica se três medidas podem formar um triângulo.
- [x] `CalculoDesconto` - Aplica desconto percentual condicionado ao valor do produto.

### 🟠 Nível 3: Estruturas de Repetição (for / while)
*Foco em loops, iterações e automação de processos repetitivos.*
- [x] `ContagemRegressiva` - Exibe uma contagem regressiva na tela de 10 a 1.
- [x] `Tabuada` - Gera a tabuada completa de 1 a 10 do número informado.
- [ ] `SomaDosNumeros` - Acumula a soma de números digitados até o usuário digitar 0.
- [ ] `Fatorial` - Calcula o fatorial de um número inteiro.
- [ ] `NumerosPrimos` - Identifica se um número é primo.
- [ ] `ApenasOsImpares` - Filtra e exibe os números ímpares em um intervalo de 1 a 50.
- [ ] `ValidadorSenha` - Bloqueia a execução até que a senha correta seja inserida.

### 🔴 Nível 4: Vetores e Matrizes
*Foco em estruturas de dados em memória e manipulação de coleções.*
- [ ] `ListaDeNumeros` - Armazena valores em um vetor e os exibe na ordem inversa.
- [ ] `MaiorEMenorVetor` - Identifica os valores extremos dentro de uma coleção.
- [ ] `BuscaNoVetor` - Realiza uma busca linear de strings dentro de um vetor de nomes.
- [ ] `MediaTurmaVetor` - Calcula médias coletivas utilizando vetores de notas.
- [ ] `MatrizSimples` - Cria e soma elements de uma estrutura bidimensional (3x3).

### 🏆 Nível 5: Desafio Final (Simulação de Sistemas)
*Exercícios complexos que unem todos os pilares anteriores simulando cenários reais de mercado.*
- [ ] `SistemaLoginSimples` - Sistema com limite de 3 tentativas de acesso antes do bloqueio.
- [ ] `SimuladorCaixaEletronico` - Algoritmo para menor distribuição possível de cédulas bancárias.
- [ ] `MiniSistemaEstoque` - Aplicação controlada por menu no terminal para CRUD em memória.

## ✒️ Como executar o projeto
1. Faça o clone do repositório:
```bash
   git clone https://github.com/vanderson-andrade/exercicios-de-logica-java.git
``` 

2. Abra a pasta do projeto utilizando o IntelliJ IDEA.

3. Navegue até a pasta src/ onde estão localizadas as classes principais.

4. Execute a classe desejada clicando no botão de Run (Seta verde) na IDE.
